# SAT Brothers Website

A professional website for SAT Brothers, a premier SAT prep service on Cape Cod.

## Pages

1. **Home (index.html)** - Welcome page with hero image of a student studying
2. **Our Approach (philosophy.html)** - Details about SAT improvement philosophy with side sections for about me content
3. **Packages & Contact (packages.html)** - Pricing packages and contact information
4. **Custom Request Form (contact-form.html)** - Form for parents to request longer-term tutoring options

## Features

- Professional, clean design
- Responsive layout for all devices
- Navigation bar on all pages
- Header with SAT Brothers branding and logo placeholder
- Package pricing displayed clearly
- Custom tutoring request form
- Contact information section

## Setup

1. Replace `logo.png` with your actual SAT Brothers logo
2. Update contact information in `packages.html` as needed
3. Deploy to your web hosting service
